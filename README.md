\# PlantGuard



PlantGuard is an end-to-end crop disease classification project using PyTorch.



\## Current Progress



\- Kaggle API dataset download script

\- Train/validation/test split generation

\- Custom PyTorch Dataset class

\- DataLoaders with image augmentation



\## Planned Features



\- Baseline CNN / EfficientNet training

\- Class imbalance experiments

\- Evaluation with accuracy, macro F1, and confusion matrix

\- GradCAM explainability

\- ONNX export

\- FastAPI inference API

\- Docker deployment
